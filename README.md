# AICTE Week 3 Task 1 - Hyperparameter Tuning and Optimization

## Overview

This project demonstrates hyperparameter tuning and optimization of a machine learning model using GridSearchCV.

## Dataset

The Iris dataset from Scikit-learn was used for this task.

## Model Used

Random Forest Classifier

## Hyperparameters Tuned

* n_estimators
* max_depth
* min_samples_split

## Method

GridSearchCV with 5-fold cross-validation was used to identify the best combination of hyperparameters.

## Files Included

* task1_hyperparameter_tuning.ipynb
* task1_hyperparameter_tuning.py
* hyperparameter_tuning_report.md
* README.md

## Conclusion

This task helped in understanding the importance of hyperparameter tuning and its impact on machine learning model performance.
