# Hyperparameter Tuning and Optimization Report

## Objective

The objective of this task was to improve the performance of a machine learning model through hyperparameter tuning.

## Dataset

The Iris dataset from Scikit-learn was used. It contains measurements of iris flowers belonging to three different species.

## Model Used

Random Forest Classifier

## Hyperparameters Tuned

The following hyperparameters were tuned:

* n_estimators
* max_depth
* min_samples_split

## Tuning Method

GridSearchCV was used with 5-fold cross-validation to evaluate different combinations of hyperparameters and identify the best-performing model.

## Results

The tuning process produced the best combination of hyperparameters for the Random Forest Classifier. The optimized model achieved high accuracy on the test dataset.

## Conclusion

Hyperparameter tuning improved the model selection process and helped identify the most effective parameter configuration. This task provided practical experience in optimizing machine learning models using GridSearchCV.
